### 📏 SEO 写作要求

#### 关键词使用规范:
- **主关键词**: 文章中出现3-5次，自然分布
- **次要关键词**: 出现2-3次
- **长尾关键词**: 在小标题和内容中自然融入
- **避免堆砌**: 优先考虑读者体验

#### Meta 标签要求:
```html
<title>主关键词 + 价值主张 + 品牌名 (60字符以内)</title>
<meta name="description" content="包含主关键词的价值描述 (160字符以内)" />
```

#### 内容质量标准:
- **字数**: 每篇2000-3500字
- **原创性**: 100%原创内容，基于实际使用经验
- **实用性**: 每段都要有可操作的指导
- **权威性**: 引用游戏官方信息和社区最佳实践
- **时效性**: 内容与最新游戏版本保持同步

### 🖼️ 视觉内容要求

#### 图片规范:
- **格式**: WebP 优先，PNG 备选
- **尺寸**: 最大宽度1200px，保持16:9或4:3比例
- **压缩**: 文件大小 < 200KB
- **Alt文本**: 包含关键词的描述性文本
- **数量**: 每2-3段配一张图，重要步骤必须配图

#### 视频内容:
- **GIF动图**: 关键操作步骤制作GIF
- **截图序列**: 完整流程的分步截图
- **对比图**: 前后效果对比
- **标注说明**: 重要区域添加箭头或标注

### 🔗 内链策略

#### 内链规划:
- **每篇文章**: 至少3-5个内链
- **相关性**: 链接到相关主题文章
- **锚文本**: 使用自然的描述性文字
- **用户价值**: 真正对读者有帮助的相关内容

#### 内链网络:
```
主页 ← → 教程首页 ← → 具体教程
  ↓         ↓           ↓
工具页 ← → 进阶技巧 ← → 案例分析

```
